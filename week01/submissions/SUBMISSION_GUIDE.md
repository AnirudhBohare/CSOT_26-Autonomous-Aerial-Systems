## Common Mistakes

| Mistake | Fix |
|---|---|
| Screenshot of MATLAB home, not Library Browser | Press Ctrl+Shift+L in Simulink |
| `.slx` file missing — only screenshots submitted | Submit the actual Simulink file |
| Trim results not included | Add `trim_results_YOUR_NAME.md` with the table |
| Model is a flat block diagram (no subsystems) | Group related blocks with Ctrl+G |
